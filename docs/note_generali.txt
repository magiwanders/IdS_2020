Il codice delle classi consegnate per il primo parziale è stato modificato,

          viene più clonata per intero)


2. CRITERI TEST JUnit

Le classi di test sono state eseguite in quest'ordine:
  1. CommentTest.java
  2. DiscussionTest.java
  3. CredentialsListTest.java
  4. ServerTest.java
  5. ConnectionTest.java

L'ordine è stato definito dal fatto che Connection.java è la classe più complessa,
quindi mi risultava comodo testare tutte le classi sia a valle che a monte per
non dover creare stub/drivers complessi.
